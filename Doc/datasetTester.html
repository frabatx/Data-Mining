datasetTester module¶

Dataset generator This module creates a dataset generated by trees. The dataset is structured as follows:

  • Create a seed, a list of lists that contain n random integer.
  • Fills the seed with noise
  • Builds a tree from a sequence
  • Adds noise to the tree by adding random nodes
  • Generates the dataset

datasetTester.dataset_generator(minTree, maxTree, n_list, max_seq_value, n_sequences, tree_noise, random_trees)[source]¶
Dataset generator. Once the parameters have been set, a dataset can be built.
  • A sequence is generated
  • The sequence is then filled with noise (other random numbers)
  • A tree is generated
  • Noise is added to the tree
  • A new record is built
  • Random trees are added (noise)
  • A new record is built
Parameters:
  • minTree (Integer) – Minimum number of trees generated per sequence
  • maxTree (Integer) – Maximum number of trees generated per sequence. Has to be greater or equal than minTree
  • n_list (Integer) – Sequence lenght
  • max_seq_value (Integer) – Number of elements inside a list
  • n_sequences (Integer) – Number of sequences generated
  • tree_noise (Integer) – Number of nodes to add as noise in a tree
  • random_trees (Integer) – Number of random trees between trees created with a sequence (noise)
